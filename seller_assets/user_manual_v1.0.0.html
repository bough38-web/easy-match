Easy Match v1.1.1 가이드

함수 없이 1분 만에 끝내는 스마트 데이터 매칭

🛡️ 윈도우 보안 경고 시 필독

프로그램 실행 시 "Windows의 PC 보호" 또는 "게시자를 확인할 수 없습니다"라는 파란색 창이 뜨나요? 이는 검증되지 않은 개인 개발자 앱에 대한 윈도우의 기본 방어 기제입니다.

해결 방법:
1단계: 파란색 경고창에서 [추가 정보] 글자를 클릭하세요.
2단계: 하단에 나타나는 [실행] 버튼을 누르면 정상적으로 시작됩니다.

🚀 업무가 3배 빨라지는 꿀팁

Easy Match는 "현재 열려있는 엑셀 인식" 기능을 지원합니다.

  • 파일을 매번 폴더에서 찾지 마세요.
  • 작업 중인 엑셀 파일을 그냥 열어둔 상태에서 Easy Match를 켭니다.
  • '현재 열려있는 워크북 인식' 메뉴를 클릭하면, 불러온 파일 리스트가 즉시 나타납니다.

01. 기본 사용법

[프로그램 메인 인터페이스]

  1. 파일 로드: 기준 파일(Source)과 대상 파일(Target)을 드래그하여 넣습니다.
  2. 항목 선택: 두 파일이 공통으로 가진 항목(사번, 이름, ID 등)을 매칭 키로 지정합니다.
  3. 데이터 체크: 대상 파일에서 기준 파일로 가져오고 싶은 열(전화번호, 주소 등)을 선택합니다.
  4. 매칭 시작: 하단의 버튼을 클릭하면 즉시 데이터가 결합됩니다.

02. 프로그램 각부 명칭 및 기능

Easy Match 프로그램의 주요 화면 구성 요소와 기능에 대한 설명입니다.

[프로그램 주요 기능 영역]

  • ① 기준 파일 (Source File)

    - 3가지 방법으로 파일을 불러올 수 있습니다:
      1. 클릭하여 찾기: 영역을 클릭하여 폴더에서 파일을 선택합니다.
      2. 드래그 앤 드롭: 엑셀 파일을 마우스로 끌어와서 이 영역에 놓습니다.
      3. 열린 시트 인식: 이미 엑셀 파일이 열려있다면, 상단 메뉴의 [현재 열려있는 워크북 인식]을 클릭하여 즉시 불러옵니다.

  • ② 대상 파일 (Target File)

    - 대상 파일도 기준 파일과 동일하게 클릭, 드래그 앤 드롭, 열린 시트 인식 3가지 방법으로 불러올 수 있습니다.
    - 기준 파일의 데이터와 매칭할 전체 데이터가 담긴 파일입니다.

  • ③ 매칭 키 설정 (Matching Key)

    - 두 파일을 연결할 공통 고유값(예: 사번, 전화번호, ID 등)을 선택합니다.
    - Tip: 중복되지 않는 고유한 값을 사용할수록 정확도가 높아집니다.

  • ④ 가져올 데이터 선택 (Output Data)
    - 기본 선택: 가져오고 싶은 컬럼의 체크박스를 선택합니다.
    - 고급 필터 기능 (조건부 값 가져오기):
    • 값 로드 (화살표 -> 버튼): 해당 버튼을 누르면 그 컬럼에 존재하는 고유한 값(중복 제거)들을 리스트로 불러와 선택할 수 있습니다.
    • 조건 설정 (IF 함수 기능): 특정 조건에 맞는 데이터만 가져오거나 필터링할 수 있습니다.
      예) 월정료 >= 30,000 (3만원 이상인 경우만)
      예) 지역 = "서울" (서울 지역 데이터만)
    • 다중 조건: 여러 컬럼에 조건을 걸면 모두 만족하는(AND) 데이터만 매칭됩니다.
  • ⑤ 실행 및 결과 (Action & Result)

    - [매칭 시작] 버튼을 눌러 작업을 수행합니다.
    - 작업 완료 후 결과 요약(성공/실패 건수)이 표시되며, 결과 파일을 저장할 수 있습니다.

    [매칭 결과 미리보기 화면]

03. 데이터 치환 (Data Replacement)

매칭 전, 대상 파일(Target)의 특정 데이터를 원하는 값으로 자동 변경하는 기능입니다.
(예: "강원본부" -> "강북/강원본부"로 통일하여 매칭)

  1. 메뉴 실행: 상단 메뉴의 [도구] - [데이터 치환 규칙 설정]을 클릭합니다.
  2. 규칙 추가:
    • 대상 컬럼명: 값을 변경할 엑셀 컬럼 헤더 이름을 정확히 입력합니다.
    • 변경 전 (Old): 바꾸고 싶은 기존 값을 입력합니다.
    • 변경 후 (New): 최종적으로 변경될 값을 입력합니다.
    • [추가/수정] 버튼을 눌러 리스트에 등록합니다.
  3. 프리셋 저장 (선택): 자주 쓰는 규칙은 상단의 [저장 (Save)] 버튼을 눌러 이름을 지정해 두면, 다음에 불러와서 사용할 수 있습니다.
  4. 적용: [닫기 (적용)] 버튼을 누르면 설정된 규칙이 매칭 시 자동으로 적용됩니다.
주의: 치환 규칙은 매칭 작업(Join)이 수행될 때 적용됩니다. 원본 파일 자체를 수정하지는 않습니다.

04. 다중 파일 일괄 매칭 (Advanced Batch)

수십 개의 파일을 일일이 매칭할 필요 없이, 여러 대상 파일을 한 번에 불러와 기준 데이터에 통합(Merge)할 수 있습니다.

[다중 파일 선택 및 설정 상세 화면]

  • ① 다중 선택 실행

    - 대상 데이터 영역의 [다중 선택] 버튼을 클릭하여 전용 관리 창을 엽니다.
    - [+ 파일 추가]를 눌러 매칭할 모든 파일들을 목록에 넣습니다.

  • ② 파일별 상세 설정
    - 시트/헤더: 파일마다 데이터가 시작되는 시트와 헤더 위치를 개별 지정할 수 있습니다.
    - 매핑 (Mapping): 파일의 컬럼명이 기준 데이터와 다를 경우, 매핑 버튼을 눌러 이름을 일치시킵니다.
    - 컬럼 선택: 각 파일에서 결과물에 포함하고 싶은 항목만 골라낼 수 있습니다. (설정됨: 컬럼(n) 표시)
  • ③ 통합 컬럼 선택 (Convenience)

    - [통합 컬럼 선택] 버튼을 누르면 모든 파일에 공통으로 존재하는 컬럼들을 한 번에 골라 일괄 적용할 수 있어 매우 편리합니다.

  • ④ 병합 매칭 수행

    - [적용 (Apply)]를 누른 후 메인 화면에서 [매칭 실행]을 클릭하면, 모든 파일의 데이터가 기준 파일 우측으로 순차적으로 결합됩니다.

04. 자주 묻는 질문 (FAQ)

Q: 데이터 매칭이 0건으로 나옵니다.

A: 매칭 키로 설정한 데이터가 서로 정확히 일치하는지 확인해 주세요. (예: 공백 포함 여부 등) 이지매치는 자동 공백 제거 기능을 제공하므로 대부분의 경우 문제없이 작동합니다.

Q: 개인 정보가 유출될 걱정은 없나요?

A: 이지매치는 모든 작업을 고객님의 **내부 PC**에서만 수행합니다. 어떠한 데이터도 외부 서버로 전송하지 않으므로 안심하고 사용하셔도 됩니다.

Q: 맥(macOS)에서도 사용 가능한가요?

A: 네, 전용 Mac 버전을 설치하신 후 '마우스 우클릭 -> 열기'를 통해 동일하게 사용하실 수 있습니다.
